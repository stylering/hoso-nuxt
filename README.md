# hoso-nuxt

> NUXT构建的服务端渲染，兼容SEO

> NUXT SSR开模式下的：移动端适配、滑动加载、图片延迟加载

> 货色HOSO — 只做原创品牌

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
